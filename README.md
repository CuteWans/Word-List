# Word-List
一、
1. 从网上找到一本英文读物，要求至少20页。
2. 从网上找到停词表。

二、实现链表类。

三、以所找到的英文读物为输入文件，统计词频，并记入链表中。链表中的每个结点形式如下：
1. 按单词的首字母建立不同的链表，即所有以a开头的单词都在第一个链表中，所有以b开头的单词都在第二个链表中，依此类推。
2. 所有链表的表头指针保存在一维数组中。
3. 读入文件时，同一个链表以字典序排序。读入完毕，按词频降序重排各链表。
4. 使用停词表，不统计停词表中的单词。
5. 不区分大小写。不统计数字等。

输入：

——《The Adventures of Sherlock Holmes》in book.in

——stopwords.in

输出：

——output.txt
